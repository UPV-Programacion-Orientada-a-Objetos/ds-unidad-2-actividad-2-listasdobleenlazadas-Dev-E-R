Detailed Description

Lista doblemente enlazada implementada manualmente para almacenar el mensaje decodificado.

Esta clase implementa una lista doblemente enlazada desde cero (sin STL) que mantiene el orden de los caracteres decodificados del protocolo PRT-7.

Note
Prohibido el uso de std::list o cualquier contenedor STL
