Detailed Description

Trama que contiene un carácter a decodificar.

Las tramas LOAD representan fragmentos de datos (L,X) donde X es un carácter. Al procesarse, el carácter se mapea usando el estado actual del rotor y se agrega a la lista de carga.

Ejemplo de trama serial: "L,H" -> letra='H'
