Detailed Description

Trama que modifica la rotación del rotor de mapeo.

Las tramas MAP representan instrucciones de rotación (M,N) donde N es un entero. Al procesarse, el rotor gira N posiciones, cambiando el mapeo de todas las tramas LOAD subsecuentes.

Ejemplos de tramas seriales:

  • "M,2" -> rotar +2 posiciones (A->C)
  • "M,-2" -> rotar -2 posiciones (A->Y)
